About JekyllSnippets.com

JekyllSnippets.com aims to provide super-awesome resources, tutorials, link-lists and of course snippets related to Jekyll to give you the best way to start developing your own websites with Jekyll. Because Jekyll comes built-in with Github Pages we also think it is a great way to publish while having everything open sourced. This page is using Jekyll, too.

JekyllSnippets.com was founded by Kevin Gimbel to provide a page to hold resources and information about Jekyll, the blog-aware static-site generator by Tom Preston-Werner, Nick Quaranto, and many more awesome contributors.

This site, however, is not related to the Core Jekyll Team, Github or any officials. It's a Fan-made project.
